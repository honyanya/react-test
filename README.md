# react-test

React.js を触ってみた

* version
```bash
$ node -v
v7.10.1
$ npm -v
4.2.0
```

* setup
```bash
### npm でローカルインストールしたコマンドを使えるようにするためのパス設定
$ echo 'export PATH=./node_modules/.bin:$PATH' >> ~/.your_profile

$ cd ~/
$ git clone https://github.com/honyanya/react-test.git
$ cd react-test
$ npm install
```
